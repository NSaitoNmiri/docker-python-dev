# docker-python-dev
a docker project for python development environment based on ubuntu (xenial)

## How to use
1. build image

    ```
    make build
    ```

1. start container

    ```
    make up
    ```

1. attach container using bash shell

    ```
    make exec-bash
    ```

1. stop container

    ```
    make stop
    ```

1. restart container

    ```
    make start
    ```

1. stop and delete container

    ```
    make down
    ```


1. list existing containers

    ```
    make ps
    ```
